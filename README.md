This project is built with **Node.js** and **TypeScript**, and **GraphQL**.

## Setup Instructions

The .env file is given and includes database connection URL and the port.

The backend is under the api folder.
Once you are placed under the api folder run the following commands.

1. Install dependencies

```
npm install
```

2. Run the development server

```
npm run dev
```

3. Build for production

```
npm run build
npm run start
```
